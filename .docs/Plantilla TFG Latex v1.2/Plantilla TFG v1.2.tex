\chapter*{Resumen}
\addcontentsline{toc}{chapter}{\numberline{}Resumen}

El presente Trabajo de Fin de Grado aborda el diseño y la implementación de \textit{The Show Verse}, una aplicación web avanzada de gestión y descubrimiento de películas y series que permite a los usuarios unificar todo el contenido en un solo lugar. La plataforma cuenta con secciones personalizadas para cada usuario gestionadas a través de APIs externas.

La dispersión del contenido entre las distintas plataformas de \textit{streaming} genera una experiencia de usuario fragmentada y dificulta el seguimiento del contenido consumido. \textit{The Show Verse} soluciona este problema ofreciendo una plataforma que unifica el contenido de las principales páginas de puntuaciones y plataformas de \textit{streaming}.

El trabajo consiste en el desarrollo completo de una aplicación web en Next.js que integra las APIs de TMDb, Trakt.tv, OMDb, JustWatch y Plex. Se ha implementado un flujo de autenticación completo para permitir la sincronización del historial, los favoritos, la lista de pendientes, las estadísticas y las valoraciones del usuario.

Desde el punto de vista de la arquitectura, la solución combina Server Components para la carga inicial eficiente de datos y Client Components para la interacción en tiempo real, junto con API Routes propias que actúan como capa intermedia para normalizar respuestas, gestionar errores y aplicar estrategias de optimización. A nivel de diseño, se ha construido una interfaz \textit{responsive} orientada a distintos dispositivos y con navegación adaptativa.

Entre las funcionalidades más relevantes destacan el control detallado del visionado de series, permitiendo marcar episodios como vistos y mostrar de forma automática cuánto se ha avanzado en cada temporada, la consulta de valoraciones por episodio y la sincronización con Trakt y TMDb para que acciones como puntuar, guardar en favoritos o añadir a pendientes se reflejen tanto en la aplicación como en la cuenta externa del usuario.

La plataforma incorpora además enlaces externos para acceder a los recursos en otras plataformas, un sistema con varios modos de visualización en todas las listas y también se integran contenidos de comunidad, como comentarios y listas públicas, que facilitan descubrir recomendaciones de otros usuarios. En conjunto, estas capacidades convierten a \textit{The Show Verse} en una plataforma unificada, escalable y centrada en la usabilidad para la gestión del contenido personal.
