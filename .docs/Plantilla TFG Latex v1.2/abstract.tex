
This Final Degree Project presents the design and implementation of \textit{The Show Verse}, a web application for the management and discovery of films and TV series that allows users to unify all audiovisual content in a single platform. The application integrates the TMDb, Trakt.tv and OMDb APIs for metadata and personal tracking, JustWatch for streaming availability, and Plex for access to content on a local media server. A complete OAuth 2.0 authentication flow with Trakt.tv has been implemented, together with a multi-level cache strategy that reduces external API calls by up to 80\% in list views. The result is a responsive web platform with fluid animations, three display modes (Grid, List and Compact), and Lighthouse scores of 92/100 in performance and 100/100 in SEO.
