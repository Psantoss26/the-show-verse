This Final Degree Project addresses the design and implementation of \textit{The Show Verse}, an advanced web application for managing and discovering films and TV series that allows users to bring all content together in a single place. The platform includes personalized sections for each user, managed through external APIs.

The dispersion of content across different \textit{streaming} platforms creates a fragmented user experience and makes it difficult to track consumed content. \textit{The Show Verse} solves this problem by offering a platform that unifies content from the main rating websites and \textit{streaming} platforms.

The work consists of the full development of a Next.js web application that integrates TMDb, Trakt.tv, OMDb, JustWatch, and Plex APIs. A complete authentication flow has been implemented to enable synchronization of the user's history, favorites, watchlist, statistics, and ratings.

From an architectural perspective, the solution combines Server Components for efficient initial data loading and Client Components for real-time interaction, together with custom API Routes that act as an intermediate layer to normalize responses, handle errors, and apply optimization strategies. At the design level, a responsive interface has been built for different devices, with adaptive navigation.

Among the most relevant features are detailed TV-series watch tracking, allowing episodes to be marked as watched and automatically showing progress in each season, consultation of per-episode ratings, and synchronization with Trakt and TMDb so that actions such as rating, saving to favorites, or adding to watchlist are reflected both in the application and in the user's external account.

The platform also incorporates external links to access resources on other platforms, a system with several display modes across all lists and community content integration, such as comments and public lists, which helps users discover recommendations from others. Overall, these capabilities make \textit{The Show Verse} a unified, scalable, and usability-focused platform for personal content management.
