
This Final Degree Project presents the design, development, and evaluation of \textit{The Show Verse}, a web application aimed at solving the fragmentation of audiovisual information across multiple streaming, rating, and tracking platforms. The main objective is to provide users with a single environment where they can discover films and TV series, organize personal activity, and consult relevant metadata without switching between services.

The solution is implemented with Next.js and combines server-side rendering strategies with interactive client-side components. Its architecture integrates external providers through dedicated API routes that normalize responses, handle errors, and optimize request orchestration. The platform connects to TMDb, Trakt.tv, OMDb, JustWatch, and Plex, and includes OAuth 2.0 authentication flows to synchronize user actions such as watch history, ratings, favorites, and watchlist management.

From a functional perspective, the system offers episode-level progress tracking, per-episode rating visualization, validated external links, personalized artwork selection, and multiple list visualization modes (Grid, List, and Compact), together with responsive navigation across desktop and mobile devices. To improve perceived performance and scalability, the project applies a multi-layer caching strategy (ISR, server memory cache, and browser session caching) with request deduplication, reducing external API traffic by approximately 70--80\% in recurrent list views.

The resulting application demonstrates that a unified, user-centered audiovisual management platform can be built with a modular and scalable architecture, while maintaining good usability and performance metrics in real usage scenarios.
